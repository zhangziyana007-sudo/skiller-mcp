# Changelog

## v2.2.0 — 流畅度深度优化 (2026-04-09)

### 智能失效策略
- **脏标记替代数据清空** — 增删操作后不再清空所有视图数据，只标记为"脏"
- **惰性刷新** — 切换到对应视图时才检测脏位并刷新，当前视图保持不变不闪烁

### 乐观更新
- **本地删除** — 直接从内存数组 filter 掉，不再 await 重新拉取 `/api/skills`
- **上传/下架** — 只更新对应卡片的"已上传/未上传"徽章，不重建整个列表
- **GitHub 导入** — 直接 push 到本地数组并渲染，后台异步刷新缓存
- **移除阻塞调用** — 导入后不再同步 `await api('/api/community/refresh')`

### 增量 DOM 更新
- **精准节点操作** — 所有列表项添加 `data-skill` 属性，用 `CSS.escape` 精准定位
- **删除动画** — 移除单项时只操作对应 DOM 节点，带 0.2s 滑出透明动画
- **徽章局部更新** — 上传/下架只修改对应卡片的 `.upload-badge`，带缩放弹性动画

### 导航悬停预加载
- **hover 预取** — 鼠标悬停导航项时后台预取该视图数据
- **防抖保护** — 5秒内同一视图不重复预取

### 骨架屏 + 视图切换动画
- **shimmer 骨架屏** — 加载数据时显示动态渐变骨架占位，替代旋转图标和空白
- **fadeSlideIn** — 所有视图切换加 0.2s 入场动画（透明度+位移）

### 搜索防抖
- **150ms debounce** — 本地仓库、GitHub 社区、社区广场的搜索框全部加防抖
- 快速输入时只触发最后一次列表渲染，减少无效重绘

### API 请求去重
- **并发合并** — 同一 URL 的并发 fetch 请求自动合并为一个 Promise
- 快速切换视图或重复操作时避免冗余网络请求

## v2.1.0 — UI 全面优化 + 版本更新系统 (2026-04-09)

### 导航与布局
- **社区广场置顶** — 社区广场移至导航栏第一位，成为默认首页
- **分类管理内嵌** — 从独立导航项移入「我的本地仓库」模块，通过过滤栏按钮触发，在右侧面板展示
- **订阅源管理重命名** — 改为「订阅外部GitHub仓库」，语义更清晰
- **我的GitHub社区纯化** — 仅管理自己的可写仓库，不再混显外部订阅源

### 技能卡片
- **卡片化列表** — 本地仓库和GitHub社区的技能列表改为带阴影的卡片样式
- **描述预览** — 每个卡片显示1-2行描述截断，优先显示用户自定义描述
- **主题色高亮** — 卡片选中/hover时使用当前主题色渲染边框和指示条

### UI/UX 统一
- **全局圆角 24px** — 所有按钮、标签、导航项、输入框统一为 border-radius: 24px
- **主题色变量** — 新增 `--theme-active` / `--theme-active-light` / `--theme-active-text` CSS 变量，所有 active 状态颜色自动跟随主题
- **主题按钮文字化** — 从圆形 emoji 按钮改为「主题 + 当前主题名称」文字形态
- 六个主题（蜡笔小新/简洁/暗黑/海洋/森林/日落）全部适配新变量

### 版本更新系统
- **版本徽章** — 标题栏右侧显示当前版本号和 commit hash
- **自动检查更新** — 页面加载时对比本地/远程 commit，有新版本时绿色脉冲提示
- **更新详情面板** — 点击版本徽章展示更新日志、GitHub Release 信息
- **一键更新** — 执行 git pull + npm install + tsc 编译，支持重启服务
- **网络容错** — GitHub 不可达时友好提示，不阻塞页面正常使用

## v2.0.0 — 本地仓库管理重构 (2026-04-09)

### 重大变更
- **「我的本地 Skill 仓库」** — 全新双栏布局取代旧版技能浏览 + 上传/投稿分离设计
  - 左栏：搜索、状态/分类过滤、统计条、技能列表
  - 右栏：详情信息、分类管理、描述编辑、操作按钮组、子技能树/原始内容
- **侧边栏简化** — 移除分类导航树，改为 5 个清晰入口（本地仓库、社区、分类管理、MCP 管理、使用日志）
- **社区 tab 精简** — 移除「上传/投稿」tab，上传功能统一到本地仓库管理

### 新功能
- **DESCRIPTION.md 分离存储** — 上传时用户描述独立存为 DESCRIPTION.md，不再追加到 SKILL.md
- **社区卡片优先显示自定义描述** — 浏览社区技能时优先读取 DESCRIPTION.md，回退到 SKILL.md frontmatter
- **点击查看全文** — 社区技能卡片只显示描述摘要，点击名称或「查看全文」按钮加载 SKILL.md 完整内容
- **认证支持** — 订阅私有仓库获取 DESCRIPTION.md 时带 token 认证

### 移除
- `renderSkills`、`renderSkillCard`、`showSkillDetail` 本地技能旧 UI
- `renderCommunityUpload` 及相关辅助函数
- `renderCategoryNav`、`showCategory` 侧边栏分类导航
